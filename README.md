# THE SILVERY TAY INC. Website

This is a basic website for THE SILVERY TAY INC., a company providing remote virtual reception and assistant services for properties in Australia.

## Features
- Company information
- Services overview
- List of properties with staff details
- Contact information
- Company logo in header
- Responsive design

## Files
- `index.html`: Main HTML structure
- `styles.css`: Basic styling
- `script.js`: Smooth scrolling for navigation
- `images/`: Folder for images (place your logo here as "the silvery tay in photo logo.jpg")

## How to View
Open `index.html` in a web browser.

To serve locally with Python (if installed):
```
python -m http.server 8000
```
Then visit `http://localhost:8000`
